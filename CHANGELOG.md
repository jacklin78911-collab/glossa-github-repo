# Changelog

## 0.5.0

- Added prompt presets for common reading tasks.
- Added follow-up cards and tree rendering for a basic question graph. 
- Added Source button with best-effort jump back to saved web passages.
- Stored card parent IDs and source locations.
- Deleting a parent card now removes its descendants.

## 0.4.0 
 
- Added editable prompt templates in the Ask Models dialog.  
- Removed source title and URL from the default prompt.
- Added optional `{{title}}` and `{{url}}` placeholders for users who want source metadata.

## 0.3.0

- Added clipboard capture for Chrome PDF viewer workflows.
- Added prompt queueing for model-page autofill.
- Added best-effort autofill for common model input boxes.

## 0.2.1

- Added Grok, Qwen / Qianwen, and Kimi model links.

## 0.2.0

- Replaced placeholder Ask Models alert with prompt generation.
- Added model-opening workflow for multiple web LLMs.

## 0.1.1

- Added manual paste capture for PDF workflows.
- Added right-click context menu capture on normal web pages.

## 0.1.0

- Initial Chrome Manifest V3 extension.
- Added selected-text capture, local cards, and side-panel UI.
